## Judul Aplikasi
Sistem Pendataan Buku Berbasis PHP Native dan MySQL

## Deskripsi Singkat
Aplikasi ini dibuat menggunakan PHP Native dan MySQL untuk mengelola data buku perpustakaan maupun buku pribadi juga bisa. Fitur yang tersedia disini meliputi menampilkan data buku, menambah, mengubah, menghapus, serta mencari data buku.

## Cara Menjalankan Aplikasi

1. Jalankan Apache dan MySQL pada XAMPP.
2. Import file `database.sql` ke phpMyAdmin.
3. Simpan folder project di dalam folder `htdocs`.
4. Buka browser.
